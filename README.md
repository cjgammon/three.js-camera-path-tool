three.js-camera-path-tool
=========================

Crude tool for building camera paths for three.js scenes.


KEY CONTROLS
------------

= – Add Path
- – Remove Path
UP/DOWN/LEFT/RIGHT – Pan Camera
OPTION + UP/DOWN – Zoom Camera
W/A/S/D – Rotate Camera

SHIFT – Moves Faster

Click Nodes to Move
Click off Node to Deselect
